# 운동 프로그램 구조 평가 가이드라인

> CoachPlan AI가 트레이너의 프로그램 구조를 평가할 때 참고하는 근거 자료.
> 각 항목에 신뢰도 레벨 표기: ⬛ RCT/메타분석 | ⬜ 전문가 합의/전통 기반

---

## 1. 저항운동 (Resistance Training)

### 1-1. 세션 구조

| 단계 | 권장 시간 | 내용 | 근거 |
|---|---|---|---|
| Warm-up | 5-10분 | 저강도 유산소 + 동적 스트레칭 + 점진적 워밍업 세트 | ⬛ ACSM 2026, RAMP 프로토콜 |
| Main | 30-60분 | 복합운동 → 단관절 → 보조운동 | ⬛ ACSM 2026, NSCA |
| Cool-down | 5-10분 | 저강도 활동 + 정적 스트레칭 | ⬛ 심리적 효과 있음, DOMS 감소 효과는 미입증 (Afonso 2021) |

### 1-2. 운동 순서 원칙

⬛ **근거 수준: 높음** (Brandao 2020, ACSM/NSCA 공통)

1. 파워/올림픽 리프트 → 복합운동 → 단관절
2. 대근육군 → 소근육군
3. 고강도 → 저강도
4. 우선 목표 근육군을 세션 초반에 배치
5. 전신 운동 시 상체/하체 교대 배치

**평가 포인트:**
- 세션 초반 운동이 근력/비대 효과 더 큼 (약 2-3회 반복 차이)
- 이 순서를 지키는 구조 → 강점으로 평가
- 역순 배치 시 → 의도적(약점 보완) vs 비의도적 구분 필요

### 1-3. 코어 운동 배치

⬛ **근거 수준: 높음** (Saeterbakken 2022 체계적 리뷰)

| 배치 | 효과 | 대상 |
|---|---|---|
| 세션 마지막 (NSCA 권장) | 복합운동 퍼포먼스 유지, 부상 위험 감소 | 일반인, 숙련자 |
| 세션 초반 (NASM 권장) | 코어 안정화 활성화 목적 | 재활, 초보자, 안정화 단계 |
| 별도 데이 분리 | 코어 전문 훈련 가능 | 5분할 숙련자 |

**평가 포인트:**
- 코어를 메인 전에 배치 → 스쿼트/데드 퍼포먼스 감소 가능성 경고
- 코어를 메인 후에 배치 → 강점으로 평가 (단, 초보자 안정화 관점에선 NASM 방식도 유효)
- 코어 없음 → 개선 제안 (숙련자 5분할이 아닌 경우)

### 1-4. 볼륨/강도 기준

⬛ **ACSM 2026 포지션 스탠드** (137개 체계적 리뷰 종합)

| 목표 | 강도 (%1RM) | 반복 | 세트 | 휴식 |
|---|---|---|---|---|
| 근력 | 70-85%+ | 1-8회 | 2-6 | 2-5분 |
| 비대 | 60-80% | 6-12회 | 3-6 | 1-2분 |
| 근지구력 | <60% | 15-25회 | 2-3 | 30초-1분 |
| 파워 | 30-60% (경량) / 80-100% (중량) | 1-6회 | 3-5 | 2-5분 |

**평가 포인트:**
- 2-3세트면 충분 (ACSM 2026)
- 완전 실패 불필요, RIR 2-3 권장
- 주당 근육군별 10-15세트 (중급), 15-20+ (고급)

### 1-5. 빈도

| 수준 | 빈도 | 구성 |
|---|---|---|
| 초보 | 주 2-3회 | 전신 |
| 중급 | 주 3-4회 | 상하체 분할 또는 Push/Pull |
| 고급 | 주 4-5회 | 분할 루틴 (각 근육군 주 2회+) |

### 1-6. 점진적 과부하

⬛ 2-for-2 Rule (NSCA): 마지막 세트에서 목표 반복 +2회를 2회 연속 달성 시 부하 증가
- 상체/소근육: 2-5% 증가
- 하체/대근육: 5-10% 증가

### 1-7. 워밍업 세부

⬛ **정적 스트레칭 주의** (McCrary 2023)
- 60초 이상 정적 스트레칭 → 근력/파워 일시 감소
- 30초 미만은 영향 없음
- 워밍업에는 동적 스트레칭 권장, 정적은 쿨다운에 배치

### 1-8. 주기화

⬛ **ACSM 2026 업데이트:** 복잡한 주기화가 비주기화 대비 우위 없음 (일반인 기준)
- 핵심은 **일관성 + 점진적 과부하**
- 고급 선수에게는 블록/비선형 주기화 여전히 유효

## 2. NASM OPT 모델

⬜ **근거 수준: 전문가 합의** (개별 원리는 근거 있으나, 전체 모델의 RCT 검증은 제한적)

### 5단계 구조

| 단계 | 목표 | 세트 | 반복 | 템포 | 휴식 | 강도 |
|---|---|---|---|---|---|---|
| 1. 안정화 지구력 | 자세 조절, 관절 안정성 | 1-3 | 12-20 | 4/2/1 | 0-90초 | 50-70% |
| 2. 근력 지구력 | 안정화 + 원동근 강화 | 2-4 | 8-12 | 4/2/1 | 0-60초 | 70-80% |
| 3. 근비대 | 최대 근성장 | 3-5 | 6-12 | 2/0/2 | 0-60초 | 75-85% |
| 4. 최대근력 | 최대 힘 생산 | 4-6 | 1-5 | X/0/X | 3-5분 | 85-100% |
| 5. 파워 | 최소 시간 내 최대 힘 | 3-5 | 1-10 | X/0/X | 3-5분 | 30-45%/85-100% |

### NASM 세션 순서 (특이사항)

1. 폼롤링(SMR) → 2. 정적스트레칭(과활성근) → 3. 동적워밍업 → 4. **코어** → 5. 밸런스 → 6. SAQ → 7. 저항운동 → 8. 쿨다운

**NSCA와의 차이:** NASM은 코어를 저항운동 **전**에 배치 (안정화 활성화 목적)

## 3. 크로스핏 (CrossFit)

⬜ **근거 수준: 방법론 기반** (프로그래밍 자체의 RCT는 제한적)

### 세션 구조 (~60분)

1. **브리핑** (5분): 워크아웃 설명, 스케일링 옵션
2. **일반 워밍업** (5-10분): 모노스트럭처럴(로잉/바이크/러닝) + 동적 모빌리티
3. **스킬/근력 파트** (10-15분): 기술 연습, 바벨 점진적 워밍업
4. **WOD** (10-25분): AMRAP, For Time, EMOM 등
5. **쿨다운** (5분): 저강도 + 스트레칭

### 스케일링 원칙

- **의도한 자극을 보존**하는 방향으로 스케일링
- 3축: 부하, 볼륨(반복), 동작 난이도

## 4. 요가 (Yoga)

⬜ **근거 수준: 전통 기반 + 생체역학 원리**

### 세션 구조 (빈야사/하타, ~60-90분)

1. **센터링/프라나야마** (5-10분): 호흡 인식
2. **워밍업/태양경배** (10-15분): 수리야 나마스카라 A, B
3. **스탠딩 포즈** (15-20분): 전사, 삼각, 밸런스 — 열 생성
4. **피크 포즈 시퀀스** (10-15분): 가장 도전적인 자세로 빌드업
5. **바닥/좌위 포즈** (10-15분): 힙 오프너, 트위스트, 전굴 — 다운
6. **샤바사나** (5-10분): 최종 이완

### 시퀀싱 원칙

- 단순 → 복잡 → 단순
- 강한 후굴 후 카운터 포즈 (예: 휠 후 차일드 포즈)
- 척추 모든 방향 워밍업: 굴곡, 신전, 측굴, 회전
- 피크 포즈 메소드: 목표 자세의 선행 가동성/근력 빌드업

## 5. 필라테스 (Pilates)

⬜ **근거 수준: 효과 입증됨, 프로그램 구조는 전통 기반**

### 현재 과학적 상황

| 항목 | 상태 |
|---|---|
| 효과 (요통, 균형, 유연성) | ⬛ 메타분석으로 입증 (Owen 2022, Zhu 2023) |
| 운동 순서/시퀀싱 | ⬜ 과학적 검증 없음 (전통/교육 기반) |
| 강도 설정 (스프링) | ⬜ 표준화된 강도 척도 없음 |
| 주기화 모델 | ⬜ 연구 없음 |
| Mat vs Reformer | ⬛ 대부분 유사한 효과, 리포머가 근력/파워에 약간 우위 (PLOS ONE 2025) |

### 세션 구조 (Mat, ~50-60분)

1. **센터링/호흡** (5분): 측면 흉곽 호흡, 골반저 활성화
2. **워밍업** (10분): 골반 틸트, 척추 관절 운동, 레그 슬라이드
3. **메인** (30-35분): 클래식 순서 또는 테마별 구성
4. **쿨다운/스트레칭** (5-10분): 머메이드, 스파인 트위스트

### 진행 원칙 (Polestar/BASI)

- 초보: 앙와위 → 복와위 → 측와위 → 좌위 → 입위
- 지지면 축소, 사지 부하 추가, 레버 길이 증가로 진행
- 클래식 순서 (34개 동작): The Hundred → Roll Up → ... → Push-Up

### 주요 연구 결과

- **요통**: 주 1-2회, 60분 미만, 3-9주 프로그램이 최적 (Owen 2022 네트워크 메타분석, n=9,710)
- **균형 (노인)**: 동적 균형에 중간-큰 효과 크기, 낙상 감소 효과는 미입증 (Moreno-Segura 2024)
- **자세**: 주 2회, 50-60분 세션에서 자세 정렬 개선 (Batista 2024)
- **운동선수**: 자세 안정성, 민첩성, 코어 근력, VO2 max 향상 (Yilmaz 2023)

### 평가 시 참고사항

필라테스 프로그램의 구조를 평가할 때:
- 운동 순서의 "정답"이 과학적으로 정해져 있지 않음을 인지
- 생체역학적 원리 (척추 방향별 워밍업, 앙와위→입위 진행)로 타당성 평가
- 주당 빈도, 세션 시간, 대상 집단 적합성은 근거 기반으로 평가 가능
- "이 순서가 최적이다"보다는 "이 구조가 대상에게 적합한가"로 접근

## 6. 공통 평가 기준 (모든 모달리티)

### 구조 강점으로 볼 수 있는 패턴

- 워밍업 → 메인 → 쿨다운 3단계 존재
- 점진적 강도 증가 (저→고→저 아크)
- 대상(초보/중급/고급)에 맞는 볼륨/강도 설정
- 운동 순서가 목표에 부합 (근력: 복합 우선 / 안정화: 코어 우선)
- 구체적 세트/횟수/휴식 명시

### 구조 개선이 필요한 패턴

- 워밍업 또는 쿨다운 누락/미기재
- 운동 순서가 목표와 불일치 (예: 근력 목표인데 단관절 먼저)
- 대상 불일치 (예: 초보자에게 5분할 구조)
- 볼륨 과다/부족 (ACSM 기준 대비)
- 코어 배치가 의도 없이 비효율적 위치에 있음
- 특정 근육군/움직임 패턴 편중
